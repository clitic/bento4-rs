A collection of crates that provide a high-level API built on top of [Bento4](https://github.com/axiomatic-systems/Bento4)'s `ap4` library.

## Crates

- [bento4-src](https://github.com/clitic/bento4-rs/tree/main/bento4-src) - Source code of Bento4 and logic to ap4 library.
- [mp4decrypt](https://github.com/clitic/bento4-rs/tree/main/mp4decrypt) - Decrypt CENC protected MP4 content using Bento4.
